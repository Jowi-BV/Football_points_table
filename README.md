This code receives a list of dictionaries with information about teams from a professional soccer league.

It calculates and organizes information such as:
- Total points
- Goal Difference
- Wins
- Loses
- Draws
- Position

Then, it organizes each team based on their total points and finally, it shows a table with all the recopilated information,
giving data such as the teams which are quialified to competitions like Champions League, Conference League, Europa League and even the ones relegated.
